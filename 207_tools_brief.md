# 🛠️ Security Sprint — Бриф инструментов

## Что у вас есть

- Репозиторий с уязвимым FastAPI-приложением
- Docker Desktop
- Python 3.12+
- Доступ в интернет

## Что нужно найти

В проекте есть уязвимости следующих типов:

| Тип | Чем искать |
|-----|-----------|
| 🔑 Секреты в коде | Secret Detection |
| 📦 Уязвимые зависимости |  SCA |
| 🐛 Уязвимости в коде |  SAST |
| 🐳 Уязвимости в Docker-образе | Image Scanning |

## Инструменты (минимум 2)

Выберите **минимум два** инструмента из списка. Все работают на Windows.

### 1. Gitleaks — поиск секретов

Находит захардкоженные пароли, API-ключи, токены, приватные ключи.

```bash
# Установка Windows (Scoop)
scoop install gitleaks

# Запуск
gitleaks detect --source=. -v

# Через Docker (если Scoop недоступен)
docker run --rm -v ${PWD}:/path zricethezav/gitleaks:latest detect -v --source=/path
```

**Что искать:** AWS keys, Stripe keys, пароли БД, JWT secrets, API keys.

---

### 2. Trivy — анализ зависимостей и образов

Универсальный сканер. Два режима:

**Режим 1 — Filesystem (SCA):** сканирует зависимости проекта
```bash
# Установка Windows (Scoop)
scoop install trivy

# Сканирование зависимостей
trivy fs --severity CRITICAL,HIGH,MEDIUM .
```

**Режим 2 — Image:** сканирует Docker-образ
```bash
# Сборка образа
docker build -t taskflow-api .

# Сканирование
trivy image taskflow-api --severity CRITICAL,HIGH
```

**Что искать:** CVE в пакетах, уязвимости в базовом образе, проблемы конфигурации.

---

### 3. Bandit — статический анализ Python-кода

Находит SQL-инъекции, использование небезопасных функций, проблемы с обработкой данных.

```bash
# Установка
pip install bandit

# Запуск (подробный режим)
bandit -r . -ll

# Только высокий и критический уровень
bandit -r . -ll -iii
```

**Что искать:** SQL-инъекции, использование `eval()`, `exec()`, `subprocess`, MD5 для паролей.

---

### 4. Semgrep — универсальный статический анализ

Альтернатива Bandit. Работает с любым языком.

```bash
# Установка
pip install semgrep

# Запуск с авто-конфигом
semgrep --config auto .
```

**Что искать:** Те же проблемы, что и Bandit, плюс дополнительные паттерны.

---

## Рекомендуемая последовательность

```
1. Gitleaks      → найдёт секреты (быстро, 1-2 мин)
2. Trivy fs      → найдёт CVE в зависимостях (5-10 мин)
3. Bandit        → найдёт уязвимости в коде (2-5 мин)
4. Trivy image   → найдёт проблемы в Docker-образе (5-15 мин)
```

## Как документировать находки

Для каждой уязвимости запишите на Яндекс.Доску:

```
🔴/🟡/🟢 [Критичность] | [Краткое описание]
📍 Файл: [путь], строка [N]
🔧 Инструмент: [название]
📝 Описание: [1-2 предложения]
```

**Критичность:**
- 🔴 Critical / High — 1-2 балла
- 🟡 Medium — 0.5 балла
- 🟢 Low — 0.25 балла

## Полезные команды для ручного анализа

```bash
# Поиск секретов вручную
findstr /s /i "password" *.py
findstr /s /i "api_key" *.py
findstr /s /i "secret" *.py

# Проверка версий пакетов
pip list

# Просмотр содержимого Docker-образа
docker run --rm taskflow-api ls -la /app
```
